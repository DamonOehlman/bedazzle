Bedazzle

CSS3 Transforms and Transitions from JavaScript

Bedazzle is a JavaScript library which makes it simple to animate DOM elements through CSS3 Transitions and Transforms.

Simple Command Syntax

Bedazzle uses a simple, chainable command syntax:

// animate the x position from 0 to 100
bedazzle('.box')
  .move('x', 100)
  .run('go');

Feel Free to be Terse

Bedazzle doesn't mind if you prefer typing less characters:

// move the box from 0 to 100
bedazzle('.box', 'x100 go');

Stateful by default

CSS Transforms aren't stateful (nor should they be), but bedazzle is:

// move the box right, and then move and rotate
bedazzle('.box')
  .move('x', 100)
  .run('go', function() {
      this.move('x', 100)
          .move('r', 180)
          .run('go');
  });

And the same again, but terse:

// move the box right, and then move and rotate (again)
bedazzle('.box', 'x100 go x100 r180 go');

Undo Things

Want to go back? Well, you can:

// move as before, but then undo
bedazzle('.box', 'x100 opacity0.5 go x100 r180 go undo go undo go');

Customize Transitions

Bedazzle assumes a 1s linear (boring). You can change this by passing options:

// bedazzle with easing
bedazzle(
  '.box',
  'x100 go x100 r180 go undo go undo go', {
      transition: '0.5s ease-in-out'
  }
);
